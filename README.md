# SupportPulse: Customer Support Load Forecasting & Staffing Analysis

## Project Overview
Support operations teams often struggle with unpredictable ticket volumes, leading to understaffing during peak demand and inefficiencies during low demand.

This project analyses historical support ticket data to identify demand patterns, peak load windows, and operational risk periods.

## Project Goals
- Identify hourly and daily ticket demand patterns
- Detect backlog and SLA pressure periods
- Compare ticket creation vs resolution trends
- Provide operational insights for staffing decisions
